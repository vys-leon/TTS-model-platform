# TTS-model-platform
 Веб-приложение на Streamlit для озвучки текста с помощью open-source ИИ-моделей.

## Описание приложения
 С помощью данного приложения можно с помощью выбранной модели озвучить введённый текст. Можно озвучить голосом по умолчанию, либо кастомным голосом, благодаря функции клонирования голоса по короткой записи голоса.

 ## Инструкция по запуску
 Для начала склонируйте к себе данный репозиторий:
 ```bash
git clone https://github.com/vys-leon/TTS-model-platform.git
```

*Примечание: Все команды ниже необходимо запускать из корня проекта:* `./TTS-model-platform/`  

Затем установите необходимые библиотеки
```bash
pip install -r requirements.txt
```

Запустите приложение streamlit
```bash
streamlit run ai_voice_platform.py
```

Готово! Приложение теперь доступно по адресу: `localhost:8501`
 ## Технический стек
 - Python
 - Streamlit
 - f5-tts, soundfile
